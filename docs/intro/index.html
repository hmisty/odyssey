🎁 坐标：新手村 - 着陆点空投​

信息

任务名称：和组长取得联络，领取新手空投礼包（包含：JTI身份认证 + 0.017 J 燃料）。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 通关建议
欢迎进入元码链的世界！你现在已经安全着陆在新手村。
现在，你需要首先领取到新手空投礼包。包含：JTI可信身份认证标识一枚（灵魂绑定NFT），以及 0.017 J 的免费燃料（gas）。
  1. 联系刘教链，加入内测群。
  2. 与10位组长中任意一位取得联络，请组长安排你加入分组群。
  3. 安装小狐狸钱包，保管好自己的助记词，配置好元码链网络，记下自己的元码链地址。（此步骤遇到困难请在分组群中交流）
  4. 向组长提供必要的信息，请组长申请发放新手空投礼包。
  5. 查看小狐狸钱包，确认燃料到账。
  6. 【可选】打开元码链浏览器，搜索自己的地址，打开自己的地址页面，查看地址剩余的燃料数量，以及JTI可信身份认证标识。
  7. 任务完成！
  1. ✅ 区块链钱包的安装和使用；
  2. ✅ 区块链钱包的网络设置；
  3. ✅ 区块链浏览器的使用。
  1. 据链友反馈，谷歌浏览器插件版小狐狸钱包、Firefox浏览器插件版小狐狸钱包、手机版小狐狸钱包App、手机版imtoken钱包App可以完成任务所需操作；
  2. 如何上网、如何安装，苹果手机海外id，等问题，请在分组小群里向其他链友请教。

🌍 坐标：新手村 - 飞翔的J​

信息

任务名称：领取创世徽章「飞翔的J」。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 通关建议
现在，你已经领取到新手免费燃料（gas）了。接下来，你需要首先掌握必要的web3生存技能——合约交互，到链上领取创世徽章——一枚灵魂绑定NFT——「飞翔的J（Flying J）」。
  1. 打开小狐狸钱包（注：浏览器插件版）的高级设置，开启”十六进制数据”功能。
  2. 用小狐狸钱包向「飞翔的J」合约地址 0x045b997b5e05df9795985ab9e6720d94557255be 发起交易，金额 0 J，附加十六进制数据 0x4e71d92d 。
  3. 等待交易上链完成，用小狐狸钱包的“在浏览器中查看账户”功能，打开元码链浏览器查看自己钱包中已经领取到的「飞翔的J」徽章。
  4. 【可选】把元码链浏览器中展示的「飞翔的J」编号和徽章截图，发到内测群展示。
  5. 任务完成！
  1. ✅ 区块链合约交互；
  2. ✅ 区块链浏览器的使用。
  1. 据链友反馈，谷歌浏览器插件版小狐狸钱包、Firefox浏览器插件版小狐狸钱包、手机版imtoken钱包App可以完成任务所需操作；手机版小狐狸钱包App可能暂不支持十六进制数据功能。
  2. 遇到问题和困难，请在分组小群里向其他链友请教。

❤️ 坐标：新手村 - JNS拍卖会​

信息

任务名称：拍到自己心仪的JNS域名（至少1个），并绑定到自己的链地址。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 常见问题
  • 成交排行榜
获得自己心仪的JNS藏品，成为JNS DAO的一员。
（注：JNS是J Name Service的缩写，是一种域名NFT。所有的JNS域名均以 .j 为后缀。）
  • 参与JNS拍卖会，获得自己拥有的JNS域名。（拍卖参与方法）
  • 【可选】在元码链浏览器中查看收到的JNS拍品并截图，发到内测群展示。
  • 【可选】拍卖可以多次参与，获得多个中意的域名NFT藏品。
  • 把你的JNS和你的地址绑定。（绑定方法）
  • 【可选】尝试和其他小伙伴交换你们的JNS藏品。
  • 任务完成。
  1. ✅ 社群互动；
  2. ✅ NFT的使用和交换。
  1. 问：JNS是永久有效的，还是（像互联网域名那样）需要定期续费的呢？
    答：JNS是链上NFT，所有权属于持有者，无需定期续费即可永久持有。
  2. 问：JNS的原始版权及授权协议是怎样的？
    答：JNS的原始版权属于JNS项目。授权协议初步考虑借鉴BAYC的极大化授权方式，把包括商业使用权在内的诸多权利都赋予JNS的持有者。
  3. 问：为什么每场拍卖只拍5个名字（不多拍一些呢）？
    答：实践表明，拍卖5个名字就需要耗时1-1.5小时，如果再多，时间更长，大家都会过于疲劳。
  4. 问：为什么不把预约列表中非热门名字直接免拍过，仅留下热门名字参拍呢？
    答：原因和上面类似，热门拍品需时较长，冷门则较短，搭配着来，时间比较合适；如果全是热门，时间会过长。
  5. 问：为什么限制每人每次只能提名预约一个名字，而不是放开无限量预约呢？
    答：为了防止预约列表过长，增加其他拍友阅读和识别、记忆上的困难。

❤️ 坐标：新手村 - JNS DAO​

信息

任务名称：参与JNS DAO的提案讨论和投票表决，获得POAP徽章（每个人的徽章颜色都不相同哦）。

  • 任务描述
  • 通关指南
  • 点亮技能
  • 常见问题
参与JNS DAO的提案讨论和治理投票，获得投票纪念徽章JNSVote POAP。
  • 积极参与到JNS DAO的日常讨论和任务中。参与或推动提案的讨论和表决。获得POAP徽章。
  • 任务完成。
  1. ✅ DAO的参与和贡献；
  2. ✅ 民主投票和社群自治启蒙。
  1. 问：JNS DAO的治理流程？
    答：三段式治理。前两步是链下的民意调查和共识投票，最后一步是链上投票。
  2. 问：什么是JNSVote POAP？
    答：POAP全称Proof-Of-Attendance Protocol，又称参与证明。JNSVote POAP是你参与JNS DAO链上治理投票的区块链证明。JNSVote POAP是灵魂绑定NFT。
  3. 问：JNS DAO链上投票表决的规则？
    答：这就要具体参见《JNS DAO基本法》了。据我所知，JNS DAO常委会正在主持起草相关内容。
